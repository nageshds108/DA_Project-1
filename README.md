# Vendor Performance Analytics

## Project Overview
Vendor Performance Analytics is a data-driven project aimed at evaluating, monitoring, and improving vendor efficiency using analytical techniques and interactive dashboards. The project focuses on understanding vendor behavior, delivery performance, cost efficiency, and overall contribution to business operations through exploratory data analysis, detailed analytical modeling, and visual reporting.

This project demonstrates practical skills in **data analytics, Python-based analysis, and Power BI dashboarding**, making it suitable for academic evaluation as well as industry-facing portfolios.

## üß© Project Files Description

### EDA.ipynb (Exploratory Data Analysis)
This notebook focuses on the initial exploration and understanding of the vendor dataset.

Key objectives covered in this file include:
- Loading and inspecting raw vendor-related data
- Understanding data structure, data types, and schema
- Handling missing values and identifying inconsistencies
- Analyzing distributions of numerical variables
- Identifying patterns, trends, and potential outliers
- Performing basic statistical analysis to gain early insights

This notebook lays the foundation for further analysis by ensuring data quality and revealing meaningful insights that guide business-focused analytics.

---

### Vendor_perf_Analysis.ipynb (Detailed Vendor Performance Analysis)
This notebook performs in-depth analysis to evaluate vendor performance using business-oriented metrics.

Major analysis performed:
- Vendor-wise performance comparison
- Cost and spending analysis across vendors
- Delivery efficiency and reliability evaluation
- Identification of high-performing and underperforming vendors
- Use of aggregations, grouping, and calculated metrics
- Data transformations for meaningful reporting

This notebook translates raw data into actionable insights that can support decision-making such as vendor selection, contract optimization, and risk management.

---

### VendorPerf.pbix (Power BI Dashboard)
This Power BI file provides an interactive and visual representation of vendor performance metrics.

Dashboard highlights:
- Vendor-wise KPIs and performance indicators
- Interactive filters and slicers for dynamic analysis
- Trend analysis and comparative visuals
- Clear and business-friendly charts and tables
- Visual storytelling for quick executive insights
